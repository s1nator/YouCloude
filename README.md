# YouCloude

YouCloude — это универсальная утилита и библиотека Python для работы с облачными хранилищами **Яндекс.Диск** и **Dropbox**. Проект предоставляет как удобный интерфейс командной строки (CLI) для пакетной работы, так и гибкий API для интеграции в ваши собственные приложения.

## Особенности

*   **Мультиплатформенность**: Поддержка Яндекс.Диска и Dropbox через единый интерфейс.
*   **Пакетный режим**: Удобный CLI для выполнения операций с файлами.
*   **Красивый вывод**: Структурированный и читаемый листинг файлов хранилища.
*   **Отказоустойчивость**: Встроенные механизмы обработки ошибок обеспечивают надежность выполнения операций.
*   **Параллельная работа**: Поддержка многопоточности для ускорения загрузки и скачивания (настраивается в конфиге).
*   **Programmer-friendly**: Понятная архитектура классов и абстракций, позволяющая легко использовать код проекта в сторонних разработках.

## Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/YouCloude.git
    cd YouCloude
    ```

2.  Установите необходимые зависимости (убедитесь, что у вас установлены `requests`, `dropbox`, `pyyaml`):
    ```bash
    pip install -r requirements.txt
    ```

## Конфигурация

Все настройки хранятся в файле `configuration.yaml`. Перед началом работы укажите ваши токены доступа и пути:

```yaml
database:
  token_for_yandex_disk: "ВАШ_YANDEX_TOKEN"
  access_token_for_dropbox: "ВАШ_DROPBOX_TOKEN"
  limit_files_for_listing: 20
  max_workers: 4  # Количество потоков для параллельной работы
  # ... остальные пути
```

## Использование (CLI)

Запуск утилиты осуществляется через `main.py`.
Синтаксис: `python main.py [ПРОВАЙДЕР] [КОМАНДА] [АРГУМЕНТ]`

### Выбор провайдера
*   `-y` или `--yandex` : Работа с Яндекс.Диском
*   `-d` или `--dropbox` : Работа с Dropbox

### Команды
*   `-l` : Вывести список файлов (красивый листинг).
*   `-g <путь>` : Скачать файл с диска (`get`).
*   `-p <путь>` : Загрузить файл на диск (`put`).
*   `-pc <путь>` : Загрузить папку целиком (рекурсивно).

### Примеры

**Яндекс.Диск:**
```bash
# Получить список файлов
python main.py -y -l

# Загрузить локальный файл на диск
python main.py --yandex -p my_document.txt

# Загрузить папку с файлами
python main.py -y -pc ./my_folder
```

**Dropbox:**
```bash
# Скачать файл с Dropbox
python main.py -d -g remote_image.jpg
```

## Использование в коде (API)

Проект спроектирован с учетом принципов ООП. Базовый абстрактный класс `Cloud` (в `cloud_interface.py`) определяет единый контракт для всех провайдеров, что упрощает расширение и использование в стороннем коде.

Пример использования в вашем скрипте:

```python
from src.yandex_disc_api import YandexDiskApi
from src.config import token_for_yandex, path_for_files_yandex

# Инициализация API
cloud = YandexDiskApi(
    token=token_for_yandex,
    root_path=path_for_files_yandex,
    # ... другие параметры
)

# Использование методов
print(cloud.list_files(10))
cloud.upload_file("report.pdf")
```
